IDLgrImage::TileDataLoaded

The IDLgrImage::TileDataLoaded function method tests whether a specified image tile has data loaded.

Syntax

Result = Obj->[IDLgrImage::]TileDataLoaded( TileInfo )

Return Value

Returns 1 (one) if the tile has data already loaded, or 0 (zero) otherwise.

Arguments

TileInfo

A named structure describing the tile to be queried. Only one tile at a time can be passed to TileDataLoaded. The structure contains the following fields:

Field

Data Type

Description

X

Long

The horizontal position of the left side of the tile(s) measured from the left of the image in pixels.

Y

Long

The vertical position of the bottom of the tile(s) measured from the bottom of the image in pixels.

WIDTH

Long

The width of the tile(s) in pixels.

HEIGHT

Long

The height of the tile(s) in pixels.

LEVEL

Long

The data resolution, where 0 is the full resolution image. A value of n (n > 0) indicates that the resolution is half that of the n-1 level.

DEST

OBJREF

The object reference to the destination object that displays the tiled IDLgrImage.
