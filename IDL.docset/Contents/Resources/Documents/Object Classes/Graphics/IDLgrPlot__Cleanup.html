IDLgrPlot::Cleanup

Syntax | Arguments | Keywords | Version History

The IDLgrPlot::Cleanup procedure method performs all cleanup on the object.

Syntax

Obj->Cleanup

or

or

OBJ_DESTROY, Obj

None
