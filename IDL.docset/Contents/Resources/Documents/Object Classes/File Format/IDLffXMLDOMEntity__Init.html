IDLffXMLDOMEntity::Init

The IDLffXMLDOMEntity::Init function method initializes the object, but this class should not be instantiated directly. Instead, methods such as IDLffXMLDOMDocumentType::GetEntities coupled with IDLffXMLDOMNamedNodeMap::GetNamedItem are used to create objects of this class.

This class should not be subclassed by a user class because the classes that create objects, such as IDLffXMLDOMDocument, will not be aware of any user-defined classes.
