IDLffXMLDOMNode::GetNodeType

Syntax | Return Value | Arguments | Keywords | Version History

The IDLffXMLDOMNode::GetNodeType function method returns the subclass type of the calling node.

Syntax

Result = Obj->[IDLffXMLDOMNode::]GetNodeType()
