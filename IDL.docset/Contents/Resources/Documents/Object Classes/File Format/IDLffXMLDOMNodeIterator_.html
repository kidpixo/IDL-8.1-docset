IDLffXMLDOMNodeIterator::Cleanup

The IDLffXMLDOMNodeIterator::Cleanup procedure method performs cleanup on the node-iterator object. As part of this cleanup, the method destroys any other objects that were created by the iterator object. In particular, the node object returned when the iterator was last moved is destroyed. The destruction of the iterator object does not affect any corresponding nodes in the DOM tree.

This class should not be subclassed by a user class because the classes that create objects, such as IDLffXMLDOMDocument, will not be aware of any user-defined classes.

Syntax

Obj->Cleanup

or

OBJ_DESTROY, Obj

Arguments

None

Keywords

None
