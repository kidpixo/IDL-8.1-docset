IDLffXMLDOMDocumentFragment::Cleanup

Syntax | Arguments | Keywords | Version History

The IDLffXMLDOMDocumentFragment::Cleanup procedure method destroys both the object (that is, the accessing object in the IDL tree) and any objects created by that object. It does not modify the actual DOM tree.

This class should not be subclassed by a user class because the classes that create objects, such as IDLffXMLDOMDocument, will not be aware of any user-defined classes.

Syntax

OBJ_DESTROY, Obj

Arguments

None

Keywords

None
