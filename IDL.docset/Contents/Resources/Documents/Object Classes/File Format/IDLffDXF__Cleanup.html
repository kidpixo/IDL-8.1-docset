IDLffDXF::Cleanup

Syntax | Arguments | Keywords | Version History

The IDLffDXF::Cleanup procedure method performs all cleanup on the object.

Syntax

Obj->Cleanup

or

Obj->[IDLffDXF::]Cleanup

or

OBJ_DESTROY, Obj

Arguments

None

Keywords

None
