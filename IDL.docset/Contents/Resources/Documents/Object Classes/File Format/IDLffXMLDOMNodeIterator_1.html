IDLffXMLDOMNodeIterator::Init

The IDLffXMLDOMNodeIterator::Init function method initializes the object, but this class should not be instantiated directly. Instead, use the IDLffXMLDOMDocument::CreateNodeIterator method to create instances of this class.

This class should not be subclassed by a user class because the classes that create objects, such as IDLffXMLDOMDocument, will not be aware of any user-defined classes.
