IDLffMJPEG2000::GetProperty

Syntax | Arguments | Keywords | Version History

The IDLffMJPEG2000::GetProperty procedure method retrieves a property or group of properties for an IDLffMJPEG2000 object.

Syntax

Obj->[IDLffMJPEG2000::]GetProperty [, PROPERTY=variable]

Arguments

None

Keywords

Any property listed under IDLffMJPEG2000 Properties that contains the word “Yes” in the “Get” column of the properties table can be retrieved using this method. To retrieve the value of a property, specify the property name as a keyword set equal to a named variable that will contain the value of the property.
