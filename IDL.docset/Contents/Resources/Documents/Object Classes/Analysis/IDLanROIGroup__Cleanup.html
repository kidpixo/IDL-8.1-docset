IDLanROIGroup::Cleanup

Syntax | Arguments | Keywords | Version History

The IDLanROIGroup::Cleanup procedure method performs all cleanup for a region of interest group object.

Syntax

Obj->Cleanup

or

Obj->[IDLanROIGroup::]Cleanup

or

OBJ_DESTROY, Obj

Arguments

None.

Keywords

None.
