IDLanROI::Cleanup

Syntax | Arguments | Keywords | Version History

The IDLanROI::Cleanup procedure method performs all cleanup for a region of interest object.

Syntax

Obj->Cleanup

or

Obj->[IDLanROI::]Cleanup

or

OBJ_DESTROY, Obj

Arguments

None.

Keywords

None.
