IDLnetOGCWCS::Cleanup

Syntax | Arguments | Keywords | Version History

The IDLnetOGCWCS::Cleanup method performs all cleanup on the object. All open network connections are closed when the IDLnetOGCWCS object is destroyed.

Syntax

Obj->Cleanup

or

Obj->[IDLnetOGCWCS::]Cleanup

or

OBJ_DESTROY, Obj

Arguments

None

Keywords

None

Version History

 

6.4

Introduced
