Add Method

 

The Add method adds another item to an existing legend.

Example

plot = PLOT(/TEST, NAME='First Plot', COLOR='lime')

l = LEGEND()

plot2 = PLOT(RANDOMU(seed,200), NAME='Second Plot', $

   COLOR='purple', /OVERPLOT)

l.add, plot2

Syntax

legend.Add, TARGET

Arguments

TARGET

One or more graphic items to be added to the existing legend.
