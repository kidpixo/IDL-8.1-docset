Draw Multi-Sided Shapes

Use the POLYGON function to draw a square, rectangle, or other multi-sided shape on an existing graphic. You can use POLYGON at the time you create the graphic, or from the command line after the graphic is displayed.

The example shows an image of Mars with a shape that defines a study area.

The code shown below creates the graphic shown above. You can copy the entire block and paste it into the IDL command line to run it.

; Define the image to display.

file = FILEPATH('marsglobe.jpg', $

SUBDIRECTORY = ['examples', 'data'])

 

; Display the mars image.

mglobe = IMAGE(file, TITLE='Mars Study')

 

; Define the large study area.

coords = [[116, 298],[251, 298], $

   [268, 147], [135, 147]]

 

; Draw the large study area using POLYGON.

alpha = polygon(coords, TARGET=mglobe, /DATA, $

   FILL_BACKGROUND=0, COLOR='white', THICK=2)

 

; Draw the text for the large study area.

alphatext = TEXT(113, 307, TARGET=mglobe, $

   'Study Area $\alpha$', $

   COLOR='white', /DATA, FONT_SIZE=18)

 

Arguments and properties used in the example

IMAGE

• TITLE - defines the title for the window.

POLYGON

• coords - this variable is an array of vectors that specifies the location of the shape.
• TARGET - Use this after graphic creation to specify the graphic in which to appear. (Not needed if used during graphic creation or if only one graphic is active.)
• DATA - defines that the location coordinates are defined in the data space.
• FILL_BACKGROUND - defines the background fill, which is white by default. Setting this property to 0 turns off the background fill.
• COLOR - defines the color of the polygon line.
• THICK - defines the line thickness.

TEXT

• X, Y - these arguments define the location to appear in the graphic, in normal, data or device coordinates.
• TARGET - Use this after graphic creation to specify the graphic in which to appear. (Not needed if used during graphic creation or if only one graphic is active.)
• text string - defines the text. Define mathematical symbols and Greek letters within dollar signs ($). See TEXT for a full list of the characters available.
• COLOR - defines the color to use for the element.
• DATA - defines that the location coordinates are defined in the data space.
• FONT_SIZE - defines the font size of the text.
