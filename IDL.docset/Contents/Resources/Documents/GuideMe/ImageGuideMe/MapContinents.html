Map Boundaries

You can display map boundaries such as continents, countries, states, rivers and lakes on a map or map projection. The following examples show the basics of how to use the MAPCONTINENTS function to display these boundaries.

Polar map showing continents and countries

The code shown below creates the polar map shown above. You can copy the entire block and paste it into the IDL command line to run it.

ant_map = MAP('STEREOGRAPHIC', $

   CENTER_LATITUDE=-90, $

   CENTER_LONGITUDE=0, $

   FILL_COLOR='Light Blue')

conts = MAPCONTINENTS(/COUNTRIES, FILL_COLOR='beige')

MAP properties

• map projection- defines the map projection to use.
• CENTER_LATITUDE - defines the center latitude of the area to display.
• CENTER_LONGITUDE - defines the center longitude of the area to display.
• FILL_COLOR - the color to use to fill the projection.

MAPCONTINENTS properties

COUNTRIES - displays the outline of the countries in the map.

FILL_COLOR - defines the fill color for the area contained in the MAPCONTINENTS outline.

Limited area map showing countries and rivers

The code shown below creates the graphic shown above. You can copy the entire block and paste it into the IDL command line to run it.

; Define a map of Europe.

map = MAP('STEREOGRAPHIC', FILL_COLOR = 'Light Blue', $

   LIMIT = [30.0, -15.0, 68.0, 55.0])

 

; Add the country outlines and fill color.

mc = MAPCONTINENTS(/COUNTRIES, FILL_COLOR='beige')

 

; Add the rivers.

rivers = MAPCONTINENTS(/RIVERS, COLOR='blue')

MAP properties

• map projection- defines the map projection to use.
• FILL_COLOR - the color to use to fill the projection.
• LIMIT - defines the area of the globe to display on the map.

MAPCONTINENTS properties

• COUNTRIES - displays the outline of the countries in the map.
• COLOR - defines the fill color for the area contained in the MAPCONTINENTS outline.
• TEXT - defines labels and annotations for the graphic.
