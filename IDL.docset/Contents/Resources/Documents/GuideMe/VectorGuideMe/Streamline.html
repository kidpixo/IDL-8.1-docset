Streamline Vectors

The STREAMLINE function allows you to draw streamlines in a graphic. The following example displays streamlines of global wind vector data over a global map with continental outlines.

The code shown below creates the graphic shown above. You can copy the entire block and paste it into the IDL command line to run it.

; Read the data.

RESTORE, FILEPATH('globalwinds.dat', SUBDIR=['examples','data'])

 

; Set up the map projection, grid, and continents.

map = MAP('Equirectangular', POSITION=[0.1,0.1,0.9,0.9], $

LIMIT=[0,-160,80,-50], TITLE='Wind Streamlines')

 

; Change some map grid properties.

grid = map.MAPGRID

grid.LINESTYLE = "dotted"

grid.ANTIALIAS = 0

grid.LABEL_POSITION = 0

grid.LABEL_ANGLE = 0

grid.FONT_SIZE=14

 

cont = MAPCONTINENTS(/CURRENT, FILL_COLOR="light gray")

 

; Display the streamlines on top of the map.

stream = STREAMLINE(u, v, x, y, /OVERPLOT, $

STREAMLINE_STEPSIZE=0.05, $

RGB_TABLE=33, AUTO_COLOR=1, THICK=5)

 

Properties used in the example

MAP property

• projection - defines the map projection to use. In this case, the Equirectangular projection.
• LIMIT - specifies the boundaries of the map region (by minimum latitude, minimum longitude, maximum latitude, maximum longitude). In this example, the boundaries are the entire globe.
• POSITION - specifies the position of the graphic within the window.
• TITLE - specifies the graphic title.

MAPCONTINENTS properties

• FILL_COLOR - fills in the map boundaries.
• CURRENT - draws MAPCONTINENTS outline in the current graphic window.

MAPGRID properties

• ANTIALIAS - turn off anti-aliasing on the grid lines.
• FONT_SIZE - increase the font size.
• LABEL_ANGLE - rotate the latitude/longitude labels.
• LABEL_POSITION - move the labels to the map edge.
• LINESTYLE - change the linestyle for the grid lines.

STREAMLINE properties

• U, V, X, Y - the data variables restored by the RESTORE procedure.
• OVERPLOT - displays the data as an overlay in the current window.
• RGB_TABLE - defines the color table used to display the image. If no color table is defined, this image displays in gray scale. 
• AUTO_COLOR - enables automatic coloring of according to the distance travelled at each step.
• STREAMLINE_STEPSIZE - decrease the distance taken at each step along the streamlines.
• THICK - change the line thickness.
