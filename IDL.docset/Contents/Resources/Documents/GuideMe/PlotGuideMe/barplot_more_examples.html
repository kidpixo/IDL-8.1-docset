BARPLOT Shading and Stacking

Stacked barplot example

; Define the data set.

data1 = SIN((FINDGEN(15)+1)/15*!PI/2)

data2 = data1 + COS((FINDGEN(15))/15*!PI/2)

data3 = data2 + 0.25 + RANDOMU(1,15)

 

; Plot three bars, stacked.

b1 = BARPLOT(data1, BOTTOM_COLOR="white")

b2 = BARPLOT(data2, BOTTOM_VALUES=data1, $

FILL_COLOR='yellow', BOTTOM_COLOR="white", /OVERPLOT)

b3 = BARPLOT(data3, BOTTOM_VALUES=data2, $

FILL_COLOR='red', BOTTOM_COLOR="white", /OVERPLOT)

 

; Add a title.

b1.TITLE='$sin(\omega\pi/2) + cos(\omega\pi/2) + \nu$'

Properties used in this example

• BOTTOM_COLOR - defines the color for the bottom of each bar.
• BOTTOM_VALUES - defines the lower limit of the bars.
• FILL_COLOR - defines the bar color. If BOTTOM_COLOR is also used, the color range will display between the two colors. Use RGB codes (for example, [0, 0, 255] defines the color blue), hexadecimal codes (#0000FF), or color names (blue).
• OVERPLOT - tells IDL to display the data as an overlay in the current window.
• TITLE - add a title at the top of the graphic.

Shaded, horizontal barplot example

 

; Define the data.

data1 = SIN((FINDGEN(15)+1)/15*!PI/2)

bottom = data1+COS((FINDGEN(15))/15*!PI/2)

 

; Draw the barplot.

b = BARPLOT(data1, BOTTOM_VALUES=bottom, FILL_COLOR='red',$

  BOTTOM_COLOR='yellow', C_RANGE=[0,1], /HORIZONTAL)

 

Properties used in this example

• BOTTOM_VALUES - defines the lower limit of the bars.
• FILL_COLOR - defines the bar color. If BOTTOM_COLOR is also used, the color range will display between the two colors. Use RGB codes (for example, [0, 0, 255] defines the color blue), hexadecimal codes (#0000FF), or color names (blue).
• BOTTOM_COLOR - defines the color at the lower range of data values.
• C_RANGE - defines the low and high values between which BOTTOM_COLOR and FILL_COLOR should be shaded.
• HORIZONTAL - defines the barplot to display horizontally rather than vertically.
