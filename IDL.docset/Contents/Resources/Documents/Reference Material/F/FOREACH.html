FOREACH

Syntax | Examples | Version History | See Also

The FOREACH statement executes one or more statements for each element of a variable (which can be an array, list, or hash table). Within the FOREACH loop, the element’s value and (optionally) the key or index to which the element corresponds are available.

Syntax

FOREACH Element, Variable [, Key] DO Statement

or

FOREACH Element, Variable [, Key] DO BEGIN

   statements

ENDFOREACH

Examples

FOREACH with an Array

A simple FOREACH code example:

array = [1, 3, 5, 7, 9, 11, 13, 15]

FOREACH element, array DO PRINT, 'Element = ', element

This code sample produces the following output:

Element = 1

Element = 3

Element = 5

Element = 7

Element = 9

Element = 11

Element = 13

Element = 15

The next example uses the block structure (instead of a single statement):

PRO testForeach

 

list = LIST(77.97, 'Galactic', [2, 7, 1, 8, 2])

 

FOREACH element, list DO BEGIN

  PRINT, 'Element = ', element

ENDFOREACH

 

END

After compiling and running the procedure, it produces the following output:

Element = 77.9700

Element = Galactic

Element = 2 7 1 8 2

FOREACH where Variable is an Expression

In a FOREACH statement, Variable can be an expression as well as a literal. In this example, we create a two-dimensional array and use an expression for Variable to print out the third column elements.

The 3x3 array elements are:

0 1 2

3 4 5

6 7 8

 

arr = INDGEN(3,3)

FOREACH element, arr[2,*] DO PRINT, element

IDL prints:

2

5

8

FOREACH with a HASH Table

In this example, we create a hash table and loop through all of the key-value pairs.

planets = HASH('Mercury', 0.3e24, 'Venus', 5e24, $

'Earth', 6e24, 'Mars', 0.65e24, 'Jupiter', 1900e24, $

'Saturn', 570e24, 'Uranus', 87e24, 'Neptune', 100e24, $

'Pluto', 0.7e24)


FOREACH mass, planets, pname DO PRINT, pname, mass

IDL prints:

Mercury 3.00000e+023

Venus 5.00000e+024

Earth 6.00000e+024

Neptune 1.00000e+026

Saturn 5.70000e+026

Mars 6.50000e+023

Uranus 8.70000e+025

Pluto 7.00000e+023

Jupiter 1.90000e+027

Notice that the order in which the keys are returned is arbitrary for a hash table.
