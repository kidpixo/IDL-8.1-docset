JIDLComponentListener

Declaration

public interface JIDLComponentListener

All Known Implementing Classes:

JIDLCanvas

Description

The listener interface for receiving component events (expose, resize) on a JIDLCanvas.

The class that is interested in handling these events implements this interface (and all the methods it contains). The listener object created from that class is then registered with the JIDLCanvas using the add-IDLComponent-Listener method. The listener is unregistered with the remove-IDLComponent-Listener.

Component events are provided for notification purposes; the JIDLCanvas automatically handles component redraws and resizes internally whether a program registers an additional JIDLComponent-Listener or not. The JIDLCanvas is itself a JIDLComponent-Listener and provides default behavior for expose and resize. For an expose event, the default behavior is for the JIDLCanvas to call the IDL program’s On-Expose method. For a resize, the default is to call the IDL program’s On-Resize method.

Note that clients should not register to listen to JIDLCanvas Component-Events using a Component-Listener, preferring the JIDLComponent-Listener instead.
