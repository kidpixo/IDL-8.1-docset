Routines (by topic) > Scope Functions

Scope Functions

SCOPE_LEVEL - Returns the current routine’s scope level.

SCOPE_TRACEBACK - Returns the current interpreter call stack (the sequence of routine calls to the present point).

SCOPE_VARFETCH - Returns variables outside the current routine’s local scope.

SCOPE_VARNAME - Returns the names of variables outside current routine’s local scope.
