GRIB_GTS_HEADER

The GRIB_GTS_HEADER procedure turns GTS headers on or off.

Syntax

GRIB_GTS_HEADER, MODE 

Arguments

MODE

Set to 1 to enable GTS headers, and 0 to disable them.

Examples

See GRIB_GET_VALUESfor a general example using the GRIB routines.
