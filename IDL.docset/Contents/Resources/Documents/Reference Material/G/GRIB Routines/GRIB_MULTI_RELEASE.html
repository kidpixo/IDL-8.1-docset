GRIB_MULTI_RELEASE

The GRIB_MULTI_RELEASE procedure deletes the multi field handle and releases the memory.

Syntax

GRIB_MULTI_RELEASE, Mghandle

Arguments

Mghandle

A GRIB multi field handle.

Examples

See GRIB_GET_VALUESfor a general example using the GRIB routines.
