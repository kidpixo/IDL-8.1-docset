GRIB_MULTI_WRITE

The GRIB_MULTI_WRITE procedure writes a multi-field message to a file.

Syntax

GRIB_MULTI_WRITE, Filename, Gmhandle

Arguments

Filename

The file in which the file handle is written.

Gmhandle

The multi-field GRIB handle to be written.

Examples

See GRIB_SET for a general example using the GRIB routines.
