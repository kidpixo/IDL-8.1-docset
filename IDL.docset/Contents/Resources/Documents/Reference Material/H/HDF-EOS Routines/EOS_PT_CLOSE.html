EOS_PT_CLOSE

This function closes the HDF point file.

Syntax

Result = EOS_PT_CLOSE(fid)

Return Value

Returns SUCCEED (0) if successful and FAIL (–1) otherwise.

Arguments

fid

Point file id (long) returned by EOS_PT_OPEN.

Keywords

None

Examples

status = EOS_PT_CLOSE(fid)
