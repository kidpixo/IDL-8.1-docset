Starting the DICOM Network Services Utility in System Mode

Use the SYSTEM keyword to launch the DICOM Network Services utility in system configuration mode:

DICOMEX_NET, /SYSTEM

This provides access to the system Configuration tab.

Note: On Windows, you can also select Start > Programs > IDL x.x > DICOM Network Services and click the System button to start the utility in system configuration mode. On Macintosh, double-click DicomExNetStartup in the root IDL folder.
